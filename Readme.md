## Install

Make sure you have [Node.js](http://nodejs.org/) and [MySQL](http://www.mysql.com/) installed.

npm install
```

## 1.Import

Upload `sql-file.sql` to MySQL

## 2.Configure

Edit the `db.js` file.

## 3.Start

```sh
npm start
```

Your app should now be running on [localhost:3000](http://localhost:3000/).
